# Phonepe-Transaction-Analytics-PowerBI
An end-to-end Power BI Business Intelligence project that transforms PhonePe transaction data into actionable insights through interactive dashboards, KPI monitoring, DAX calculations, Power Query transformations, and data modeling.
# 📊 PhonePe Transaction Analytics Dashboard | Power BI

An interactive Power BI dashboard developed to analyze PhonePe transaction data, user demographics, transaction performance, and business KPIs. This dashboard enables stakeholders to monitor digital payment trends, customer behavior, and transaction growth through dynamic visualizations.

---

## 📌 Project Overview

This project provides an end-to-end analysis of PhonePe transactions using Power BI.

The dashboard helps answer business questions such as:

- How many transactions were completed?
- What is the total transaction value?
- How many users are using PhonePe?
- Which services generate the highest transaction value?
- How are users distributed by age?
- What are the month-over-month growth trends?
- How do weekend transactions compare with weekdays?

---

## 🎯 Business Objectives

- Monitor transaction performance
- Track total transaction value
- Analyze customer demographics
- Measure business growth using MoM KPIs
- Identify top-performing services
- Understand transaction patterns over time
